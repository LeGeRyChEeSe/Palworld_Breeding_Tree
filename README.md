## 🛠️ For Developers

<details>
<summary><b>🔧 Development Setup</b></summary>

### Prerequisites
- Python 3.8+
- PyQt6
- NetworkX
- Graphviz

### Quick Setup
```bash
# Clone repository
git clone https://github.com/LouisMazin/Palworld_Breeding_Tree.git
cd Palworld_Breeding_Tree

# Setup virtual environment
python -m venv venv
venv\Scripts\activate.bat  # Windows

# Install dependencies  
pip install -r requirements.txt

# Run application
python Main.py
```

### Build Executable
```bash
# Build with PyInstaller
build.bat
```

</details>

<details>
<summary><b>🏗️ Architecture Overview</b></summary>

- **Core Logic**: Graph-based breeding calculations using NetworkX
- **UI Framework**: PyQt6 with qt-material styling
- **Data Storage**: JSON-based Pal database with caching
- **Visualization**: Graphviz for tree rendering
- **Pattern**: Singleton managers with Observer pattern for UI updates

</details>
